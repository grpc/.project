# Security Policy

## Reporting Security Issues

The gRPC maintainers take security seriously. We appreciate your efforts to responsibly disclose your findings.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please contact the security team at grpc-security@googlegroups.com.

For more details, see the [gRPC security policy](https://github.com/grpc/grpc/blob/master/SECURITY.md).
